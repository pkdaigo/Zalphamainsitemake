**Prevention:** Always start demo from landing page

### Issue 2: Mock Data Not Showing
**Symptoms:** Empty dashboards or job listings  
**Fix:** Mock data is hardcoded in components - should always display  
**Check:** Ensure you're logged in as correct user type (student/employer/school)

### Issue 3: Mobile View Issues
**Symptoms:** Layout broken on phone screens  
**Fix:** Platform is mobile-responsive - try rotating to portrait mode  
**Best Practice:** Demo desktop version for investors, mobile for students

### Issue 4: Slow Loading
**Symptoms:** Pages take >3 seconds to load  
**Fix:** Clear browser cache, ensure stable internet connection  
**Note:** First load may be slower (asset downloads), subsequent navigation is instant

### Issue 5: Forms Not Submitting
